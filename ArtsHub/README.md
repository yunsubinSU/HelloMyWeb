# 🎭 ArtsHub - 공연예술 크라우드 펀딩 플랫폼
- [깃헙 페이지](https://jaewoong-hwang.github.io/ArtsHub/)

## 📌 프로젝트 개요
공연 예술 창작자와 후원자를 연결하는 펀딩 및 협업 플랫폼

- 공연 전문가들의 프로젝트 개설 및 협업 지원
- 후원자 참여 기반의 투명한 펀딩 시스템
- 커뮤니티 기능과 마이페이지로 소통 강화
