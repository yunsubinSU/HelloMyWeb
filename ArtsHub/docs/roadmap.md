## 📌 세부 로드맵 (주차별)

### 1주차 (04.07 ~ 04.13)
- 프로젝트 개요 / 사용자 정의 정리
- 기존 구현 페이지 점검 및 정리
- DB 테이블 설계 초안
- 역할 분담 확정
- Notion + GitHub 협업 세팅

### 2주차 (04.14 ~ 04.20)
- 회원가입, 로그인 기능 완성
- 전문가 / 일반 사용자 역할 분리
- 사용자 DB 연동
- 로그인 세션 처리

### 3주차 (04.21 ~ 04.27)
- 프로젝트 개설 페이지 (전문가용)
- 프로젝트 상세 페이지
- 프로젝트 목록 필터 기능

### 4주차 (04.28 ~ 05.04)
- 펀딩 참여 페이지 구성
- 리워드 선택 구조 기획 및 구현
- 펀딩 참여 테이블 설계

### 5주차 (05.05 ~ 05.11)
- 테스트 결제 API 연동 (카카오페이 등)
- 리워드 배송 정보 입력
- 후원 내역 관리 기능 구성

### 6주차 (05.12 ~ 05.18)
- 커뮤니티 기능: 후기 / 이야기 / 자유 게시판
- 글쓰기, 댓글, 좋아요 기능

### 7주차 (05.19 ~ 05.25)
- 전문가 포트폴리오 상세 페이지
- 프로젝트 검색 / 필터 / 추천 기능
- 협업 요청 기능 연결

### 8주차 (05.26 ~ 06.01)
- 전체 기능 QA
- UI 개선, 반응형 처리
- 오류 및 예외 처리

### 9주차 (06.02 ~ 06.06)
- 최종 배포 (Firebase, Vercel 등)
- 소개 페이지 / 문서 정리
- 초기 사용자 피드백 대응
